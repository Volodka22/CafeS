# Приложение-меню для сети кафе и ресторанов
Приложение помогает посмотреть меню, перед походом в кафе/ресторан, а так же сделать заказ, если пользователь уже в кафе, для этого достаточно выбрать понравившиеся блюда, нажать "оформить заказ" и навести камеру на QR-код, лежащий на столе, после чего останется только ждать официанта, наслаждаясь уютной обстановкой. Сообщения о заказе приходят на отдельное приложение.
